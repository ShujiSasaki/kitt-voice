### ε. 報酬関数の具体式

```
Reward =
  w1 * Calmar_norm
+ w2 * ProfitFactor_norm
+ w3 * CapitalTurnover_norm
+ w4 * ProfitPerHour_norm
- w5 * MaxDrawdownPenalty
- w6 * LiquidationPenalty
- w7 * OvertradePenalty
- w8 * SlippagePenalty
```

各要素:
- Calmar = AnnualizedReturn / abs(MaxDrawdown)
- ProfitFactor = GrossProfit / abs(GrossLoss)
- CapitalTurnover = TotalPositionNotional / AverageEquity
- ProfitPerHour = NetProfit / ActiveTradingHours ← Shujiさんの「時間軸に対して利益」に直結
- MaxDrawdownPenalty = max(0, MaxDrawdown - DD_limit)^2
- LiquidationPenalty = 1 if liquidation_event else 0 (Reward -= 100 * LiquidationPenalty)
- OvertradePenalty = max(0, TradeCount - TradeCount_limit)
- 「見送りを美徳化」するのではなく、低品質な売買を罰する

レバ調整 Half-Kelly:
```
KellyFraction = p - (1 - p) / b
HalfKelly = 0.5 * KellyFraction
Leverage = min(MaxLeverage, HalfKelly * ConfidenceScale * VolatilityScale)
```
